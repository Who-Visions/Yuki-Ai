# Veo 3

Veo 3 is our latest line of video generation models. This page documents the capabilities and features of Veo 3.

## Model Details

| Property | Value |
| :--- | :--- |
| **Model ID** | `veo-3.0-generate-001` |
| **Versions** | 3.0 Generate 001, 3.0 Fast Generate 001, 3.0 Generate, 3.0 Fast Generate |

## Capabilities

### Supported
*   Text to video
*   Prompt rewriting
*   Sound generation

### Not supported
*   Image to video preview
*   Extend a Veo video
*   Generate videos from the first and last frames
*   Reference image to video

## Usage types

### Supported
*   Provisioned Throughput
*   Fixed quota

### Not supported
*   Dynamic shared quota

## Technical specifications

### Video aspect ratios
*   16:9
*   9:16

### Supported resolutions
*   720
*   1080

### Supported framerates
*   24 FPS preview

### Prompt languages
*   English

## Limits
*   **Maximum API requests per minute per project:** 10
*   **Maximum videos returned per request:** 4
*   **Video length:** 4, 6, or 8 seconds
