# YUKI KNOWLEDGE BASE

## 1. Image Generation (Nano Banana Pro / Imagen 4 / Gemini 3 Image)
*   **Model:** `gemini-3-pro-image-preview` or `imagen-4.0-generate-001`.
*   **Capabilities:**
    *   **Text Integration:** Flawless text on posters, products, etc.
    *   **Style Consistency:** Can replicate styles (Pixar, Halftone, Sketch) from reference or description.
    *   **Resolutions:** 1K, 2K (Recommended), 4K.
    *   **Cinematography:** Supports specific angles (Low, High, Dutch, Over-the-shoulder) and shots (Extreme close-up, Full body).
    *   **Virtual Try-On:** (Preview feature) Can generate try-on images from person + product.

## 2. Video Generation (Veo 3.1)
*   **Model:** `veo-3.1-generate-001`.
*   **Capabilities:**
    *   **Text-to-Video:** Generate 4-8s videos from prompts.
    *   **Image-to-Video:** Animate a static image.
    *   **First & Last Frame:** Generate video interpolating between two specific frames (e.g., Summer -> Winter, Young -> Old).
    *   **Cinematic Moves:** Aerial zooms, Dolly zooms, Rack focus (focal shift).

## 3. Deep Research & Reasoning (Gemini 3 Pro)
*   **Model:** `gemini-3-pro-preview`.
*   **Capabilities:**
    *   **Thinking:** High reasoning capability for complex problems.
    *   **Multimodal:** Understands video, audio, code, and PDFs.
    *   **Context:** 1M token context window.

## 4. Chat & Interaction (Gemini 2.5 Pro)
*   **Model:** `gemini-2.5-pro`.
*   **Role:** Main conversational interface. Fast, efficient, handles standard interactions and tool routing.

## 5. Cosplay Specifics
*   **Philosophy:** "I change your outfit, not your face."
*   **Workflow:**
    1.  Understand user goal (Character, Vibe).
    2.  Select Mode (Guide, Stylist, Architect).
    3.  Generate Preview (Image).
    4.  (Optional) Animate Preview (Video).
    5.  Refine based on feedback.
