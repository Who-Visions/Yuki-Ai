# Veo 2

Veo 2 is our stable line of video generation models. This page documents the capabilities and features of Veo 2.

## Model Details

| Property | Value |
| :--- | :--- |
| **Model ID** | `veo-2.0-generate-001` |
| **Versions** | 2.0 Generate 001, 2.0 Generate |

## Capabilities

### Supported
*   Text to video
*   Image to video
*   Prompt rewriting
*   Reference image to video

### Not supported
*   Insert objects into videos preview
*   Remove objects from videos preview

## Usage types

### Supported
*   Fixed quota

### Not supported
*   Provisioned Throughput
*   Dynamic shared quota

## Technical specifications

### Video aspect ratios
*   16:9
*   9:16

### Supported resolutions
*   720

### Supported framerates
*   24 FPS

### Prompt languages
*   English

## Limits
*   **Maximum API requests per minute per project:** 10
*   **Maximum videos returned per request:** 4
*   **Video length:** 5 to 8 seconds
*   **Maximum image size (image-to-video):** 20 MB
