# Imagen 4

Imagen 4 is our latest line of image generation models. This page documents the capabilities and features of the Imagen 4 models.

## Model Details

| Property | Value |
| :--- | :--- |
| **Model ID** | `imagen-4.0-generate-001` |
| **Versions** | 4.0 Generate 001, 4.0 Fast Generate 001, 4.0 Ultra Generate 001 |

## Capabilities

### Supported
*   Image generation
*   Digital watermarking and verification
*   User-configurable safety settings
*   Prompt enhancement using prompt rewriter (preview)
*   Person generation

### Not supported
*   Image customization using few-shot learning
*   Subject customization for product, person, and animal companion
*   Style customization
*   Controlled customization
*   Instruct customization or style transfer
*   Mask-based image editing
*   Insert objects in images
*   Remove objects from images
*   Outpainting
*   Product image editing
*   Upscale images
*   Negative prompting

### Image ratios and resolutions
*   **1:1**: 1024x1024
*   **3:4**: 896x1280
*   **4:3**: 1280x896
*   **9:16**: 768x1408
*   **16:9**: 1408x768
*   **1:1**: 2048x2048
*   **3:4**: 1792x2560
*   **4:3**: 2560x1792
*   **9:16**: 1536x2816
*   **16:9**: 2816x1536

### Prompt languages

*   Hindi preview
*   Japanese preview

## Limits
*   **Maximum images returned per request (text-to-image generation):** 4
*   **Maximum image size uploaded or sent in a request (MB):** 10 MB
*   **Maximum input tokens (text-to-image generation prompt text):** 480 tokens
